Embedded Media & Socials

Some use cases may require you to validate that a site can be blocked directly while still allowing embedded content from that site through. For example, block youtube.com but allow embedded content to still function.

You can use this page to test whether embedded content still functions (or not) in this, or other, scenarios.

Privacy Mode Embed

YouTube offers an optional "Privacy Mode" when getting the embed link for a video. When selected, Google does not track the user unless the video is clicked, but the domain and URL www.youtube-nocookie.com/embed/<video-id> is used instead of the traditional www.youtube.com/embed/<video-id>
